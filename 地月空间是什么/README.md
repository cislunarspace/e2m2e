## 动力学模型

限制性三体问题（Restricted Three-Body Problem, RTBP）是研究地月空间轨道动力学的基础理论框架。在该问题中，两个主天体（如地球和月球）在相互引力作用下做圆周运动，而第三个质量可忽略的航天器在两个主天体的引力场中运动。
