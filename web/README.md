## 地月空间是什么？

地月空间是指地球静止轨道之外的地球与月球引力作用区域，空间范围从距地面36000km延伸到50万千米左右，包括月球和地月平动点轨道。地月空间与近地空间的差异主要有两点：一是距离尺度大了10多倍、体积空间大了1000多倍；二是由于受到地球与月球引力的共同作用，轨道运动从基本"二体运动"演化到复杂"三体运动"，使得近地空间确定性的经典开普勒轨道转变为地月空间不稳定的混沌轨道，由此改变了人类对轨道运动的传统认知，也从根本上决定了地月空间航天活动的特性[1]。

### 地月空间定义

地月空间是地球与月球引力共同作用的特殊空间区域，其范围从地球静止轨道（约36,000公里）延伸到地月系统的引力影响边界（约50万公里）。这一区域不仅包括月球轨道，还包括五个地月平动点（拉格朗日点）及其附近的轨道空间。

### 地月空间任务开展情况

近年来，随着各国对月球探测和深空探索的重视，地月空间任务日益增多。从早期的阿波罗计划到现代的月球探测任务，再到未来的月球基地建设，地月空间已成为国际航天竞争与合作的新前沿。当前，美国、中国、俄罗斯、欧洲、日本、印度等国家和组织都在积极开展地月空间相关任务，包括月球探测、月球资源勘查、地月空间基础设施部署等。

## 地月空间飞行器在什么轨道上运行？

地月空间飞行器的轨道设计比近地空间复杂得多，主要受到地球和月球双重引力的影响。地月空间轨道可分为两大类：

### 驻留轨道

- **地月平动点轨道**：包括L1、L2、L4、L5等平动点附近的晕轨道、近直线晕轨道等
- **月球轨道**：包括近月轨道、极月轨道、月球同步轨道等
- **远距离逆行轨道**：在月球引力影响范围内但远离月球的逆行轨道

### 转移轨道

- **地月转移轨道**：从地球到月球的转移轨道，包括直接转移和低能量转移
- **月球捕获轨道**：从地月转移轨道进入月球轨道的轨道
- **平动点转移轨道**：在不同平动点之间或平动点与月球/地球之间的转移轨道

## 地月空间科学研究前沿在哪里？

地月空间科学研究涉及多个前沿领域，主要包括：

1. **轨道动力学研究**：三体问题、限制性三体问题、混沌轨道理论
2. **空间环境研究**：地月空间辐射环境、太阳风与月球相互作用
3. **月球科学研究**：月球地质、月球资源、月球内部结构
4. **航天器技术研究**：深空导航、自主控制、长期在轨维护
5. **空间基础设施研究**：地月空间通信、导航、遥感系统
